These notebooks constitutes small hobby projects made to improve my skills with python, scikit-learn, c++ and concepts in finance/data analysis.
Due to limited upload space the data sets are not included, but can generally be acquired for free if needed, below are some notes for each notebook with objectives and data source (if external data used).

Rossmann store sales analysis

  A small project about predicting stores sales from the Rossmann data set, utilizing decision trees and proper feature engineering.
  Dataset utilized is from https://www.kaggle.com/competitions/rossmann-store-sales/data

Credit Card Fraud

  A small project working on european transaction data which has been fraud flagged and anonymized using PCA, thus reduced to 28 generic uninterpretable features. 
  Showcases the usage of linear machine learning and subsequent performance evaluation to make decision rules for when to flag as fraud and when not to. 
  Dataset utilized is from https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud/data
